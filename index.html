Sobre

Sou Advogada e Graduanda em Análise e Desenvolvimento de Sistemas.

A advocacia me possibilitou desenvolver habilidades como gestão de conflitos, resolução de problemas de clientes, trabalho com prazos e sob pressão e, sei que essas habilidades também podem somar na área de tecnologia.

Decidi passar pela transição de carreira para a área de TI pois, é uma área que eu sempre gostei muito e, acho que é o momento de buscar trabalhar nesse ramo que desperta tanto minha curiosidade e interesse.

Em 2019 eu participei do projeto chamado {Reprograma}. Um projeto que ensina programação para mulheres e, eu amei aprender Desenvolvimento Front End (HTML, CSS, JavaScript). Como projeto final, desenvolvemos um site que conta a história de várias mulheres incríveis para a história e para o mundo.

Em 2020, fiz parte da turma de Desenvolvimento Back End (NodeJS) da {Reprograma} e, aprendi a criar APIs com ExpressJS e Mongoose. Além disso, eu estou me dedicando aos estudos de React e Python.

Minhas principais stacks são:
■ Desenvolvimento Front-End: HTML5, CSS3, Bootstrap, Responsive Design, JavaScript, React.
■ Desenvolvimento Back-End: NodeJS, RestAPI, MongoDB, Mongoose, Postgres, Mysql, Sequelize, ExpressJS.
■ Deploy na AWS, Digital Ocean e Heroku.
■ Robotic Process Automation: Python e Selenium.
■ Versionamento de código: Git, GitHub e GitLab.
